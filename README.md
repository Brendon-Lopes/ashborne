# Ashborne

Terminal-based roguelike built with React and Ink.

## Prerequisites

- Node.js 20+
- npm

## Getting Started

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs the game with hot reload using `tsx watch`.

## Production

```bash
npm run build
npm start
```

Builds the project with esbuild and runs the compiled output with Node.js.

## Controls

| Key       | Action            |
| --------- | ----------------- |
| `j` / `↓` | Navigate down     |
| `k` / `↑` | Navigate up       |
| `Enter`   | Confirm selection |
